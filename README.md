# Hora do Dia

Projeto desenvolvido durante meus estudos de JavaScript.

## Tecnologias

- HTML5
- CSS3
- JavaScript

## Funcionalidades

- Exibe a hora atual.
- Altera a mensagem conforme o horário.
- Troca a imagem de acordo com o período do dia.

*Projeto desenvolvido durante meus estudos no Curso em Vídeo, com adaptações e personalizações realizadas por mim.
